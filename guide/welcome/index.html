Welcome to Stencila

Thanks for trying out Stencila. We're still in beta eta development, so it's great having people like you testing things out! Please feel free to report any bugs, or make feature suggestions, on our Github repo. Or, chat with us in our Gitter room.

Stencila is the office suite for reproducible research. We're building tools for reproducible research that have the same intuitive, visual interfaces, that you and your colllegues are used to. Software that is as easy to use as Microsoft Office and Google Docs but built from the ground up to meet today's demands for data-intensive research to be transparent, reproducible and efficient.

This is a Stencila Document. It's the first platform component that we've developed - Stencila Sheets (spreadsheets) and other offce suite components will follow soon. So, lets look at what a Stencila Document can do and what makes it special.

You can edit this document just like in a traditional word processor like Microsoft Word. We've kept the interface minimal so you can focus on authoring great content. Select some text to bring up the text toolbar. Place the cursor at the start of a paragraph or other block to change it's type e.g. paragraph to heading. Press enter to start a new paragraph or insert a table or cell.

Cells
