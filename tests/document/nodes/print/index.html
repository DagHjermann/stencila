Print directives evaluate an expression in the document's execution context and insert the text representation of the result in to the document. Here is a print of the current time: .

If there is an error in the expression (e.g syntax or logic error) the print directive will not be shown in normal mode but appears red in reveal mode: ReferenceError: foobar is not defined.

A macro is available for the print directive: ${expression}.
