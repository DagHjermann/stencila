Test for emoji

Emoji are represented in document HTML using <span data-emoji>:short_name:</span>. The leading and trailing semicolons are optional but recommended because if the document is displayed without Javascript ("static mode"), they indicate the intent of the author e.g :smile: and smiley. We use data-emoji instead of class="emoji" for consistency with other Stencila Document nodes

If the short name is unknown then it is displayed with styling, again to indicate the intent of the author e.g. :pink_unicorn:

A macro is available, a short name with underscores for spaces and surrounded by semicolons e.g. :stuck_out_tongue_winking_eye:, :rocket:. The name of the emoji can be edited in the inline toolbar. At present there is no autocomplete type list of emoji on the macro or in the toolbar.

We're using EmojiOne. Full set of emoji with shortnames here.
