This is the Stencila development server for the `web` module. It bundles and compiles Javascript and CSS on the fly for convenience during development.

Static HTML files that mimic the structure of Stencila components:

  • http://localhost:5000/static/stencil/

Dynamic components, hosted in local R, Python etc sessions and listening to port 7373, can be proxied via this server. The Javascript and CSS still gets generated on the fly but everything else gets passed on to the backend session. Make sure you add the trailing slash e.g.

  • http://localhost:5000/demo/stencils/simple-r/
