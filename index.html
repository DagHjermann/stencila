Test and demo examples

Blank: blank text and spreadsheet documents, mainly for testing.

Mini cells: text and spreadsheet documents with some code cells written in Mini, Stencila's simple embedded language.

External language cells: an article with some code cells written in external languages R, Python and SQL.

R spreadsheet: a sheet with R code in it's cells

Gene conversion errors: a sheet using data on gene name auto-conversion errors from Ziemann et al (2016) Gene name errors are widespread in the scientific literature

Gene data: a sheet with an example of gene data as used in Ziemann et al (2016).

Reproducible publication: Illustrates a publication that bundles together an article and a sheet.

These examples of Stencila documents are intended for testing and demoing. Expect bugs, missing features and general annoyances! And, when you find them, please let use know at https://community.stenci.la/ or https://github.com/stencila/stencila/issues

Some examples use external code cells (e.g. in R or SQL). To execute them you will need to have an external Stencila Host running:

  • Visit http://alpha.stenci.la?ticket=XXXXXXXX with a valid access ticket (ask us for that).
  • Use our cluster of Stencila Hosts by appending &peers=http://cloud.stenci.la?ticket=XXXXXXXX to the end of each example's URL. Ask us for a ticket.
  • Run one of Stencila's Docker images locally: 
    docker run -p 2100:2000 stencila/core
    Then append &peers=http://localhost:2100 to the end of each example's URL
  • Run one of our language packages locally. See our getting started guide
