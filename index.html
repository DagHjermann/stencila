Notes

To execute external code cells you will need to have an external Stencila Host running (using either one of Stencila's Docker images, or by installing one of our packages). Then append &peers=http://localhost:2000 to the end of the URL. See our getting started guide. Alternatively, you can append &peers=https://cloud.stenci.la?ticket=XXXXXXXX to the end of the URL. Ask us for a ticket.
